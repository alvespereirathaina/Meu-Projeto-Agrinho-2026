# AgroVision
### Projeto desenvolvido para o concurso Agrinho

 O AgroVision é uma plataforma educativa voltada à conscientização sobre a sustentabilidade no agronegócio. O projeto apresenta os principais impactos ambientais causados por práticas agrícolas convencionais, destaca soluções sustentáveis e oferece um simulador interativo que permite comparar os impactos ambientais de diferentes formas de produção nas culturas de milho, trigo e soja.

## Objetivo:

Promover a  conscientizaçãosobre a importância da sustentabilidade no campo, demonstrando como práticas agrícolas mais responsáveis podem contribuir para a preservação ambiental e para uma produção mais eficiente.

## Funcionalidades:

- Design responsivo: proporcionando boa experiência de navegação tanto em computadores  quanto em dispositivos móveis;
- Poswsui recursos de acessibilidade, incluindo ajuste de fonte e modo escuro;
- Apresentação de impactos ambientais relacionados à atividade agrícola;
- Exibição de alternativas e práticas sustentáveis;
- Simulador interativo de impacto ambiental;
- Comparação entre produção sustentável e convencional;
- Exibição dos resultados em porcentagem.

## Tecnologias Utilizadas:

- HTML5 com estrutura semântica;
- CSS3 para estilização e responsividade;
- JavaScript para interatividade e funcionamento do simulador;
- Google Fonts para tipografia;
- Imagens obtidas através do Pixabay.

## Tema Agrinho:

O projeto aborda a sustentabilidade no agronegócio, incentivando a adoção de práticas agrícolas mais conscientes e demonstrando, de forma educativa e interativa, os benefícios ambientais da produção sustentável.
